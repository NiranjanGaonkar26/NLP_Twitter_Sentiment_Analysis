# NLP_Twitter_Sentiment_Analysis

Run the code in Google Colab because it offers a wide range of pre-installed libraries for implementing data science projects. Also your code runs on Google's servers, which means you don't need to worry about hardware limitations and can run your code faster. 

The dataset is sourced from Kaggle. Find it here https://www.kaggle.com/datasets/kazanova/sentiment140
Follow the steps mentioned in this article https://www.kaggle.com/discussions/general/74235 to download the dataset directly in your colab environment.

The jupyter notebook includes annotations about the tasks performed by each cell.
